# 📚 Sistema de Gestão de Sala de Aula Online

## 📌 Visão Geral
Este projeto foi desenvolvido como parte da disciplina **Modelagem e Projetos em Engenharia de Software**, com o objetivo de conceber, modelar e justificar tecnicamente uma solução de software para gestão de sala de aula online, semelhante ao Google Classroom.

A proposta contempla práticas de **Engenharia de Software**, **Gestão Ágil**, **Modelagem de Processos**, **Arquitetura de Software** e uso de ferramentas colaborativas como **GitHub** e **Kanban**.

---

## 🎯 Objetivos
- Apoiar professores e estudantes na gestão de turmas, atividades e avaliações.  
- Facilitar a comunicação entre os envolvidos.  
- Garantir organização e rastreabilidade dos processos acadêmicos.  
- Propor uma **funcionalidade inovadora** que agregue valor ao sistema.
